# Hotel Manager REST API

REST API приложение для управления отелем. Позволяет бронировать номера, управлять базой гостей, заказывать дополнительные услуги и вести историю проживания

**Стек технологий:**
* Java 17
* Spring Framework 7.0 (Context, WebMVC, ORM, JDBC)
* Hibernate ORM 6.4.4
* PostgreSQL (драйвер 42.7.8) + Liquibase 5.0.1 (миграции БД)
* Jackson (JSON обработка)
* Apache Tomcat 10/11
* Maven

## Первый запуск
Для первого запуска необходимо создать пустую базу данных PostgreSQL с именем hotel_db. Миграции накатятся автоматически при старте приложения

В терминал надо прописать команды:
```text
dropdb -U postgres hotel_db
createdb -U postgres hotel_db
```
## Настройка файлов импорта/экспорта
Приложение использует внешнюю папку для хранения CSV-файлов. По умолчанию путь настроен на вашу домашнюю директорию (`${user.home}/hotel_data/`).
Чтобы протестировать импорт данных (DataController):
1. Создайте папку `hotel_data` в вашей домашней директории (например, `C:\Users\ВашеИмя\hotel_data\` на Windows или `~/hotel_data/` на Mac/Linux)
2. Скопируйте все `.csv` файлы из папки `sample_data` (находится в корне этого репозитория) в созданную вами папку `hotel_data`

## Запуск сервера в IntelliJ IDEA
1. Убедитесь, что у вас установлен локальный сервер Tomcat (версия 10+ или 11+).
2. В IntelliJ IDEA откройте конфигурации запуска (Edit Configurations -> `+` -> Tomcat Server -> Local).
3. Укажите путь к вашему локальному Tomcat (кнопка Configure).
4. Перейдите на вкладку Deployment, нажмите `+` -> Artifact и выберите `hotel-manager:war exploded`.
5. В поле Application context (внизу) обязательно оставьте только один слеш: `/`
6. Нажмите Run. Сервер запустится на порту 8080 (или другом, указанном в настройках вашего Tomcat).
